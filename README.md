# LifeLab Backend

Платформа для самопознания и поиска хобби. LifeLab помогает пользователям находить новые увлечения, отслеживать их влияние на самочувствие и находить полезные паттерны в повседневной жизни.

## Основные возможности
* **Умный подбор хобби:** Рекомендации на основе бюджета, геолокации, интересов и типа личности.
* **Трекинг самочувствия:** Отслеживание настроения и энергии в привязке к ежедневным активностям.
* **Аналитика и паттерны:** Поиск корреляций между образом жизни и качеством жизни.
* **Персональные эксперименты:** Платформа предлагает безопасные эксперименты для улучшения рутины.

## Технологический стек (в процессе разработки)
* **Backend:** Python, FastAPI
* **Database:** PostgreSQL
* **ORM:** SQLAlchemy
* **Deployment:** Docker

## Статус
Проект находится в активной разработке.
